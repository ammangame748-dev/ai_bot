# 🤖 بوت الذكاء الاصطناعي المتكامل للديسكورد

بوت ديسكورد متطور يجمع بين قوة الذكاء الاصطناعي ولوحة تحكم ويب سهلة الاستخدام، مصمم ليعمل 24/7 على منصة Render.

## 🚀 الميزات الرئيسية

- 💬 **دردشة AI الذكية**: تحاور مع GPT-3.5 مباشرة داخل قنوات الديسكورد.
- 🎨 **توليد الصور**: أنشئ صوراً فنية مذهلة من النصوص باستخدام DALL-E.
- 🖼️ **إزالة خلفية الصور**: أرسل أي صورة وسيقوم البوت بإزالة خلفيتها فوراً.
- 📥 **تنزيل الفيديوهات**: أرسل رابط فيديو من (Instagram, TikTok, Twitter/X) وسيقوم البوت بإرسال الفيديو كملف مباشرة.
- ⚙️ **لوحة تحكم (Dashboard)**: واجهة ويب بسيطة لتخصيص القنوات المخصصة لكل ميزة دون الحاجة لتعديل الكود.

## 🛠️ المتطلبات التقنية

المشروع مبني باستخدام:
- **Node.js** (Discord.js v14)
- **Express** (للوحة التحكم)
- **yt-dlp** (لمحرك تنزيل الفيديوهات)

## 📋 كيفية الإعداد والنشر

### 1. الحصول على المفاتيح (Tokens)
يجب توفير المفاتيح التالية في إعدادات البيئة (Environment Variables):
- `DISCORD_TOKEN`: من [Discord Developer Portal](https://discord.com/developers/applications ).
- `OPENAI_API_KEY`: من [OpenAI API](https://platform.openai.com/ ).
- `REPLICATE_API_TOKEN`: من [Replicate](https://replicate.com/ ) (لميزة إزالة الخلفية).

### 2. النشر على Render
1. ارفع الملفات (`index.js`, `package.json`, `Procfile`) إلى مستودع GitHub.
2. في Render، أنشئ **Web Service** جديد واربطه بالمستودع.
3. أضف المفاتيح المذكورة أعلاه في قسم **Environment Variables**.
4. سيقوم Render بتشغيل البوت ولوحة التحكم تلقائياً.

## ⚙️ طريقة الاستخدام
بعد تشغيل البوت، افتح الرابط الذي وفره لك Render (مثلاً: `https://your-bot.onrender.com` ).
ستجد واجهة بسيطة، قم بلصق **ID القناة** لكل ميزة في الخانة المخصصة لها واضغط **حفظ**. الآن سيعمل البوت فقط في تلك القنوات المحددة.
